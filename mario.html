How to Play

You’ll need Python and Pygame to run this game. Here’s how to set it up:

  • Step 1: Install Python 3.x from python.org.
  • Step 2: Open your terminal and install Pygame with: pip install pygame
  • Step 3: Download and unzip the game file using the button above.
  • Step 4: Navigate to the unzipped folder in your terminal and run: python mario_game_v1.0.py

Tip: Check that the image file paths in the script match your setup. Adjust the script if needed. The zip includes the game script and images.
